# 🏙️ AirBnB Data Analysis & Visualization (New York 2024)

This project performs **Exploratory Data Analysis (EDA)** and **Data Visualization** on AirBnB listings data for **New York City (2024)**.  
The goal is to uncover insights into pricing, neighborhood trends, room types, and host activity using Python.

---

## 📘 Project Overview

This project involves analyzing Airbnb listing data to understand:

- How prices vary across different neighborhoods
- Which areas have the highest listing density
- The relationship between availability, price, and reviews
- Host activity and distribution trends

---

## 🧩 Steps Followed

1. **Importing all dependencies (libraries)**  
2. **Loading datasets**  
3. **Initial exploration** (shape, datatypes, missing values)  
4. **Data Cleaning** (duplicates, missing values, formatting)  
5. **Data Analysis & Visualization** (using Seaborn & Matplotlib)

---

## 🧠 Technologies Used

| Library | Purpose |
|----------|----------|
| **Pandas** | Data manipulation & cleaning |
| **NumPy** | Numerical operations |
| **Matplotlib** | Data visualization |
| **Seaborn** | Advanced statistical plotting |

Key Insights:

Top neighborhoods by average price
Most popular room types in each area
Correlation between number of reviews and price
Availability trends across listings


Visualizations include:

Bar charts showing top expensive areas
Heatmaps of correlation between numerical features
Distribution plots for price and availability


Conclusion

This project demonstrates the use of Python for real-world data cleaning, analysis, and visualization.
It provides meaningful insights into Airbnb listings and pricing trends in New York City.
